# Westside Kings & Queens

Marcus Draney's AAU basketball program in West Valley, Utah.

## About

Westside Kings & Queens provides competitive AAU basketball training and development for high school players (15U, 16U, 17U).

## Site

Static site hosted on GitHub Pages: https://ldraney.github.io/west-side-basketball/

## Tech Stack

- Plain HTML/CSS/JS (no build step, no frameworks)
- Stripe Payment Links for tryout registration
- GitHub Pages hosting
